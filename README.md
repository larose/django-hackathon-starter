# Django Hackathon Starter

A boilerplate for Django web applications.


# Prerequisites

- Python 3.5+
- [Virtualenv](https://virtualenv.pypa.io/en/stable/)


# Getting started

```
$ make
```

Open your browser at [http://127.0.0.1:8000](http://127.0.0.1:8000)
